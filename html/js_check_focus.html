js如何判断一个元素是否获得焦点

js如何判断一个元素是否获得焦点: 可能在实际应用中需求不多，也或许使用以下方式判断过于直白，不过原理总是那么回事，下面就是一个简单的判断元素是否获得焦点的例子

代码

  1. $("#theid").click(function(){
  var act = document.activeElement.id;
  if(act=="theid" ){
    alert("获取焦点了");
  }
  else{
    alert("没有获取焦点");
  }
});
以上代码实现了我们的要求，可以判断指定元素是否获得焦点。
关于activeElement可以参阅js的activeElement简单介绍一章节。
