Tpl控件函数

系统Tpl控件函数全部位于 framework/function/tpl.func.php 文件内。

控件的概念：控件是在视图中使用的可重用单元组件。例如，日期选择器控件可生成一个精致的允许用户选择日期的日期选择器，系统提供了以下控件。

注意：该文件内所有函数使用前必须加载文件: load()→func('tpl');

tpl_form_field_category_2level()

string tpl_form_field_category_2level(string $name, array $parents, array $children, int $parentid, int $childid);

二级分类选择器

参数
$name string 表单名称
$parents array 
array  父分类集合 
eg: array(
          array('id'=>1, 'name'=>'类型1'),
          array('id'=>2, 'name'=>'类型2')
    )
$children array 
array  按父分类分组的子分类集合. 
eg:  array(
           1 => array(  // 类型1 id 
                    array('id'=>3, 'name'=>'子类1'),
                    array('id'=>4, 'name'=>'子类2'),
                    ...
                ),
           2 => array(  // 类型2 id
                    ...
                ), 
     );
$parentid int 选中的父类id
$childid int 选中的子类id
返回值

生成二级分类选择器的HTML代码。

示例
// php　代码
// 大类
$cat1 = array(
        array('id'=>1, 'name'=>'轿车'),
        array('id'=>3, 'name'=>'卡车'),
        array('id'=>5, 'name'=>'跑车'),
);
// 子类
$cat2 = array(
        // 父类 id => 子类集合
    1 => array(
        array('id'=>4,'name'=>'轿车1'),
        array('id'=>6,'name'=>'轿车2'),
    ),
    3 => array(
        array('id'=>7,'name'=>'卡车1'),
        array('id'=>8,'name'=>'卡车2'),
    ),
    5 => array(
        array('id'=>9,'name'=>'跑车1'),
        array('id'=>10,'name'=>'跑车2'),
    )
);
 
// html 代码
{php echo tpl_form_field_category_2level('car',$cat1, $cat2, $_GPC['car']['parentid'], $_GPC['car']['childid']);}

tpl_form_field_daterange()

string tpl_form_field_daterange(string $name, [array $value], [boolean $time]);

范围日期选择器

参数
$name string 表单input名称
$value array 表单input值 array('start' ⇒ 开始日期,'end' ⇒ 结束日期)，默认为当前时间
$time boolean 是否显示时间
返回值

生成日期选择器的HTML代码。

示例

1: 不显示分钟 

$birth = array(
    'start' => '2000-01-01',
    'end'   => '2011-02-02',
);
{php echo tpl_form_field_daterange('birth', $birth);}

后台获取表单方式

$birth = $_GPC['birth'];// 结构为: array('start'=>?, 'end'=>?)
 
$starttime = empty($birth['start']) ? strtotime('-1 month') : strtotime($birth['start']);
$endtime   = empty($birth['end'])   ? TIMESTAMP : strtotime($birth['end']) + 86399;

页面样式

2: 显示分钟 

{php echo tpl_form_field_daterange('work2', null, true);}

tpl_form_field_image()

注: app 端图片上传,功能有所精简. 仅有上传, 没有浏览已上传图片.

Web 端调用

string tpl_form_field_image(string $name, [string $value], [string $default], [array $options]);

App 端调用

string tpl_form_field_image(string $name, [string $value]);

图片选择与上传

参数
$name string 表单input名称
$value string 表单input值
$default string 默认缩略图)
$options array 图片上传配置信息 
// 图片上传配置信息
 
array('width'  => number， // 上传后图片最大宽度
      'height' => number， // 上传后图片最大高度
      'thumb'  => true | false; // true 或不设置, 则保存为指定 width / height 的缩略图; false: 上传图片不经任何处理.
      'global' => boolean, // true：公共图库 ，false：公众号图库。
      'tabs'   => array(   // 功能选项卡, 至少包含 browser, upload, remote 一个, 含有多个时, 只有一个可以赋值为 'active'(激活).
             'browser' => 'active',  // 浏览服务器图片     
             'upload'  => '',        // 上传图片
             'remote'  => ''         // 上传图片到微信端
      ),
      'extras' => array( 'image' => 缩略图img标签的自定义属性及属性值 ,
                         'text'  => input 标签的自定义属性及属性值  )
)
返回值

生成图片选择与上传的HTML代码。

示例
{php echo tpl_form_field_image(image);}
示例

待上传图片

已上传图片 

$options = array(
    'global'=>true, 
    'extras'=>array(
        'image'=> 'title="style=\'width: 200px\'"',
        'text' => 'title="extras_text"')
);
