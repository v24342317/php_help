文件操作函数

系统文件操作函数全部位于 framework/function/file.func.php 文件内。

注意：该文件内所有函数使用前必须加载文件: load()→func('file');

file_copy

void file_copy(string $src, string $des, array $filter);

复制指定目录下所有文件到新目录

参数
$src string 原始文件夹
$des string 目标文件夹
$filter array 需要过滤的文件类型
返回值

^Void

示例
load()->func('file');
file_copy(IA_ROOT . '/web', IA_ROOT . '/data', array('php'));

file_delete

boolean file_delete(string $file);

删除文件

参数
$file stirng 需要删除的文件名
返回值

删除成功返回 true， 失败返回 false。

示例
load()->func('file');
file_delete('test.png');

最佳实践：因为函数内部会自动加上系统附件目录，所以参数只需要提供附件名称即可。

file_image_crop

mixed file_image_crop(string $src, string $desfile, [int $width], [int $height], [int $position]);

图像裁切处理

参数
$src string 原图像地址
$desfile string 新图像地址
$width int 要裁切的宽度
$height int 要裁切的高度
$position int 开始裁切的位置， 按照九宫格1-9指定位置
返回值

如果图像裁切成功，返回 true，否则返回错误信息数组。

示例
load()->func('file');
file_image_crop(IA_ROOT . '/test.png', IA_ROOT . '/test2.png', 50, 50);

file_image_thumb

mixed file_image_thumb(string $srcfile, [string $desfile], [int $width]);

图像缩略处理

参数
$srcfile string 需要缩略的图像
$desfile string 缩略完成后的图像
$width int 缩放宽度
返回值

如果图像缩略成功，则返回缩略图的地址，否则返回错误信息数组。

示例
load()->func('file');
file_image_thumb(IA_ROOT . '/test.png', IA_ROOT . '/test2.png', 500);

file_lists

array file_lists(string $filepath, int $subdir, [string $ext], [int $isdir], [int $md5], [int $enforcement]);
参数

^$filepath | string | 目录名称 |

$subdir int 是否搜索子目录
$ext string 搜索扩展名称
$isdir int 是否只搜索目录
$md5 int 是否生成MD5验证码
$enforcement int 是否开启强制模式
返回值

扫描指定目录中的文件，如果参数 $ext 不为空，则返回对应的文件类型，最后返回由文件名组成的数组。

示例
load()->func('file');
$files = file_lists(IA_ROOT . '/web/common', 1, '.php', 0, 1);
print_r($files);

file_move

boolean file_move(string $filename, string $dest);

将文件移动至目标位置

参数
$filename string 移动的文件
$dest string 移动的目标位置
返回值

将指定文件移动到指定的目录，如果目录不存在，则创建，返回一个是否移动成功的布尔值。

示例
load()->func('file');
file_move(IA_ROOT . '/test.log', IA_ROOT . '/web/test.log');

file_random_name

string file_random_name(string $dir, string $ext);

获取指定某目录下指定后缀的随机文件名

参数
$dir string 目录的绝对路径
$ext string 文件后缀名
返回值

返回获取到的随机文件名称。

file_upload

array file_upload(array $file, [string $type], [string $name], [boolean $is_wechat]);

上传文件到附件目录

参数
$file array 上传的文件信息
$type string 文件保存类型
$name string 保存的文件名，如果为空则自动生成
$is_wechat boolean 是否上传到微信服务器
返回值

返回上传的结果，如果失败，返回错误数组，否则返回上传成功信息及保存的文件路径。

示例
load()->func('file');
file_upload($_FILE['test'], 'image', 'test.png');

file_write

boolean file_write(string $filename, string $data);

将数据写入文件

参数
$filename string 文件名称
$data string 写入数据
返回值

将数据 $data 写入到文件 $filename 中，如果文件 $filename 不存在，则创建，返回一个是否写入成功的布尔值。

示例 
load()->func('file');
file_write(IA_ROOT . '/test.log', 'hello-world');

mkdirs

boolean mkdirs(string $path);

递归创建目录

参数
$path string 需要创建的目录名称
返回值

根据参数创建对应的目录，并返回一个是否创建成功的布尔值。

示例
load()->func('file');
mkdirs(IA_ROOT . '/web/hello/world/example');

rmdirs

boolean rmdirs(string $path, [boolean $clean]);

删除目录

参数
$path string 目录位置
$clean boolean 是否删除整个目录
返回值

如果参数 $clean 为 true，则不删除目录，仅删除目录内文件; 否则删除整个目录。

示例
load()->func('file');
rmdirs(IA_ROOT . '/test');
