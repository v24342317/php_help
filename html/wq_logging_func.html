日志相关函数

日志相关函数全部位于 framework/function/logging.func.php 文件内。

注意：该文件内所有函数使用前必须加载文件: load()→func('logging');

logging_run

mixed logging_run(mixed $log, [string $type], [string $filename], [boolean $includePost]);
参数
$log mixed 日志内容
$type string 日志类型
$filename string 日志文件名称
$includePost boolean 是否记录POST信息
返回值

如果记录成功，返回 true，否则返回错误信息。

示例
load()->func('logging');
logging_run('粉丝删除成功');

logging_mkdir

mixed logging_mkdir();
参数

void

返回值

如果创建成功，返回创建完成的目录名，否则返回 false。

示例
load()->func('logging');
logging_mkdir();
